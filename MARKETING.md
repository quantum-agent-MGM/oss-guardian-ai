### Tagline

### Description
OSS Guardian AI es un agente AI que se conecta a tu repo de GitHub y revisa automáticamente cada PR. Detecta bugs de seguridad, genera comentarios profesionales (nada de "AI slop"), crea fixes, CHANGELOGs automáticos y te da un OSS Health Score semanal.

🔍 Security Auditor IA — detecta prompt injection, secretos expuestos y vulnerabilidades en <8 segundos
💬 Comentarios humanos — maintainers piensan que es una persona real
📝 Auto CHANGELOG — markdown perfecto para cada release
📊 OSS Health Score — dashboard semanal con score 0-100

Stack: GitHub App + Claude AI + Next.js. Zero config, 2 clics para conectar.

🎯 Para maintainers de proyectos open-source con más de 50 estrellas. Ahorra 10+ horas/semana.
💰 $19/mes (Starter) · $29/mes (Pro) · 7 días gratis sin tarjeta
🔗 oss-guardian-ai.vercel.app
